# Commande_Fruits_Legumes
Commandes Fruits et Légumes 🍎🍋

Bienvenue sur le dépôt du projet Commandes Fruits et Légumes ! Ce site web permet aux utilisateurs de commander des fruits et légumes frais directement en ligne, avec un panier interactif et un système de paiement sécurisé. Ce projet inclut des fonctionnalités avancées pour une meilleure expérience utilisateur.
Fonctionnalités principales 🌟

    Affichage des articles dynamiques : Les produits sont affichés sous forme de cartes avec images, détails et promotions.
    Ajout au panier avec animation : Une animation visuelle se déclenche lors de l’ajout d’un article au panier.
    Filtrage par catégories et origines : Les utilisateurs peuvent filtrer les articles par fruits, légumes ou par origine.
    Mode sombre 🌙 : Un bouton permet d'activer ou de désactiver le mode sombre.
    Pagination dynamique : Affichage des articles paginés pour éviter la surcharge d’informations.
    Recherche dynamique 🔍 : Les utilisateurs peuvent rechercher un article par nom.
    Système de promotion : Affichage des produits en promotion avec un prix barré et réduit de 20%.
    Notification de confirmation : Message de confirmation élégant lors de l'ajout au panier.
    Paiement sécurisé 💳 : Une page dédiée permet aux utilisateurs de renseigner leurs informations bancaires.
    Nom de domaine personnalisé : Déployé avec le domaine abakartech.me.

Technologies utilisées 💻

    HTML5 : Structure des pages.
    CSS3 : Design et styles personnalisés (animations, responsive design).
    JavaScript : Fonctionnalités dynamiques (AJAX, événements, animations).
    GitHub Pages : Hébergement du site.

Structure du projet 🗂️

├── index.html               # Page principale du site (accueil)
├── payement.html             # Page de paiement sécurisé
├── commande.css              # Styles pour la page principale
├── payement.css              # Styles pour la page de paiement
├── commande.js               # Script principal pour la gestion des articles et du panier
├── payement.js               # Script pour la validation du formulaire de paiement
├── CNAME                     # Fichier pour le domaine personnalisé (abakartech.me)
├── donnees/                  # Données statiques (articles.js, tarifs.js)
├── images/                   # Images des produits et icônes
└── README.md                 # Documentation du projet

Comment utiliser ce projet 🚀

    Cloner le dépôt GitHub :

    git clone https://github.com/USERNAME/REPOSITORY_NAME.git
    cd REPOSITORY_NAME

    Lancer en local :
        Ouvrez le fichier index.html dans votre navigateur.

    Modifier le contenu :
        Ajoutez vos propres images de produits dans le dossier images/.
        Ajoutez des articles personnalisés dans donnees/articles.js.

Déploiement 🌍

Le site est déployé via GitHub Pages et accessible à l'adresse : https://abakartech.me.
Étapes pour le déploiement :

    Poussez vos modifications sur le dépôt :

    git add .
    git commit -m "Mise à jour"
    git push origin main

    Votre site sera automatiquement mis à jour sur GitHub Pages.

Exemple d’utilisation 🛒

    Ajoutez des articles au panier.
    Passez la commande en cliquant sur "Payer".
    Vous serez redirigé vers la page de paiement où vous pourrez entrer vos informations bancaires.
